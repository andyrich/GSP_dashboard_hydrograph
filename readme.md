# GSP map query

## how to run:

1) make a virtual env like: 

    python -m venv venv


2) activate the environment: 

    '''venv/Scripts/activate''' for windows
    
    source venv/bin/activate' for linux
    
3) install the requirements

    pip install --r requirements.txt

4) start the application:

    python app_main.py
